# 🌊 AquaStudy: Study Smart, Save Coral - Agentic Learning for Ocean Warriors

An AI-powered agentic study platform designed to transform how students learn while raising awareness about coral reef conservation and ocean pollution. Powered by **Google Gemini API**, this free, ad-free application combines intelligent tutoring with environmental education to create meaningful impact.

## About This Project

This is our submission for the **Gemini API Developer Competition**. We've built a comprehensive agentic studying tool that intelligently adapts to students' learning needs while seamlessly integrating coral conservation awareness throughout the learning journey. Every study session becomes an opportunity to make a difference for our oceans.

**Zero Cost. Zero Ads. Zero Subscriptions.** Just pure learning with purpose.

## 🎓 Core Features Powered by Gemini AI

### **Personalized Student Dashboard**
An organized, intuitive workspace designed to keep you focused and productive. Fully customizable with themes, colors, and layout preferences to match your style.

### **Intelligent Question Generator**
Generate unlimited practice questions for any subject and unit:
- **Format Options**: Multiple Choice (MCQ) or Free Response (FRQ)
- **Difficulty Customization**: Easy, Medium, Hard, AP, Honors, or standard level
- **Smart Validation**: Gemini ensures questions are contextually relevant to your class
- **Image-Based Generation**: Upload a problem image and generate similar practice questions
- Real-time feedback and step-by-step solutions

### **Smart Flashcard Creator**
Learn more effectively with AI-generated flashcards:
- Three generation methods: Text prompts, image-based, or curriculum-aligned
- Study with built-in coral conservation facts at checkpoints
- Interactive flipping and practice modes
- Direct export to Quizlet for flexibility

### **Study Assist & Personalized Tutor**
Live, step-by-step help powered by Gemini:
- Ask questions about any worksheet problem
- Receive personalized explanations and guidance
- Choose between "Study Assist" for homework help or "Tutor" mode for concept mastery
- Learn at your own pace with patient, adaptive instruction

### **Curated Marketplace**
A Gemini-powered algorithmic recommendation engine:
- Discover study resources tailored to your learning style
- Smart search and filtering across subjects
- Community-driven content recommendations

### **Collaborative Discussion Threads**
- Ask clarifying questions within problem sets
- Get Gemini-powered responses with contextual understanding
- Community support with real-time Firebase database
- Build knowledge together

### **Real-Time Progress Tracking**
- Visual progress bars for study sessions
- Coral checkpoint achievments with fun facts and bleaching awareness education
- Track your academic growth while learning about ocean conservation

## 🌱 Ocean Conservation Integration

Every time you study, you're contributing to coral awareness:
- **Coral Facts at Checkpoints**: Learn about coral bleaching, reef ecosystems, and conservation efforts
- **Interactive Learning**: Answer questions about coral health while you progress
- **Gamified Awareness**: Unlock conservation milestones as you complete study sessions
- **Purpose-Driven Learning**: Transform passive studying into active environmental citizenship

## 🛠️ Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **AI**: Google Gemini API (core intelligence)
- **Database & Auth**: Firebase (Authentication, Firestore, Real-time Database)
- **Integrations**: Desmos, Quizlet Export
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn
- Firebase project configuration
- Google Gemini API key

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Backend setup
cd backend
npm install
# Configure your environment variables in .env

# Frontend setup
cd ../web
npm install
# Configure your Firebase and Gemini API settings in .env.local
```

### Running Locally

```bash
# Backend
cd backend
npm run dev

# Frontend (in another terminal)
cd web
npm run dev
```

Visit `http://localhost:3000` to access the application.
